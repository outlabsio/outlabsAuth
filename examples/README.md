# OutlabsAuth Examples

This directory contains example applications demonstrating each preset.

## Examples

### 1. Simple App (`simple_app/`)
Demonstrates SimpleRBAC preset:
- User registration and login
- Role assignment
- Basic permission checks
- Protected routes

**To be implemented in Phase 2**

### 2. Hierarchical App (`hierarchical_app/`)
Demonstrates HierarchicalRBAC preset:
- Entity hierarchy (Organization → Department → Team)
- Entity memberships
- Tree permissions
- Multiple roles per user

**To be implemented in Phase 4**

### 3. Full Featured App (`full_featured_app/`)
Demonstrates FullFeatured preset:
- Everything from Hierarchical
- Context-aware roles
- ABAC conditions
- Redis caching

**To be implemented in Phase 6**

### 4. Migration Example (`migration_example/`)
Shows before/after for migrating from centralized API to library.

**To be implemented in Phase 7**
